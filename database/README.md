# Database Files for Data Analysis Team

## Contents

- `scripts/`  
  SQL scripts for creating/restoring the database structure and seed data.

- `backups/`  
  SQL Server backup files used by the Data Analysis team.

## How to Use

### Option 1 — Restore Backup

Use SQL Server Management Studio:

1. Right click on Databases
2. Choose Restore Database
3. Select Device
4. Choose the `.bak` file
5. Restore

### Option 2 — Run SQL Script

Open the `.sql` file in SQL Server Management Studio and execute it.
